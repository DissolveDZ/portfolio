SpriteLight

A game engine capable of 2D and 3D PBR written in C and OpenGL

SpriteLight is my most recent game engine and I'm currently working on a rewrite in rust. This project was started around September of 2023. Using PBR and other modern rendering methods. It also features hot-reloading and highly abstracted drawing functions. The engine is also capable of rendering high-quality MSDF text on the gpu. It's running on a differed pipeline and is capable of physically-based bloom etc.
